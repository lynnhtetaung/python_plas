# A quick-demo of aggregation in Pandas
#.count(): This gives a count of the data in a column.
#.sum(): This gives the sum of data in a column.
#.min() and .max(): This helps to find the minimum value and maximum value, ina function, respectively.
''' Assume the testCSV.csv includes the following data:
         City          State         Capital      Population
0     Philadelphia   Pennsylvania     No       1.581 Million
1    Sacramento     California        Yes      0.5 Million
2      New York       New York        No       8.623 Million
3      Austin           Texas         Yes      0.95 Million
4      Miami           Florida        No       0.463 Million  '''

import pandas as pd
#read the CSV file
df = pd.read_csv(r'D:\testCSV.csv')
print(df.count())

# to find the sum of elements in a column
df.sum()

# for only one column
print(df.City.sum())

# find the minimum and maximum values of the dataset
print(df.City.min())
print(df.City.max())
